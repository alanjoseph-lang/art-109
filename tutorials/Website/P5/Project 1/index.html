Project 1

Drawing Pad

Description: For my final project, I combined all my learned skills from the previous exercises. I learned how to use the case function in order to assign different functions to different keys and toggle between them. My mouse pressed and variable knowledge was also put to the test here. My theme was pixel/sketch horror. I wanted to create a bunch of pixel and halftone brushes so that I could create dark, distorted, and shadowy pieces that could give off an ominous vibe.

  • 1 = Default Brush
  • 2 = Pixel Brush
  • 3 = Halftone Brush
  • 4 = Hatch Halftone Brush
  • 5 = Crosshatch Halftone Brush
  • 6 = Screentone Brush
  • 7 = Radial Brush
  • Z = Eraser
  • X = Clear
